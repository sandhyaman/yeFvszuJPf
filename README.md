# 医疗预约与诊断系统 python619

## 项目概述

本项目是一个基于 Python Flask 的医疗预约与诊断系统，涵盖了管理员、医生和用户三种用户角色。系统旨在提供便捷的在线预约挂号、诊断病历管理、在线咨询等功能，以优化医疗服务流程。

## 技术栈

- 开发语言：Python
- Web 框架：Flask
- 数据库：MySQL

- 用户管理
- 医生管理
- 科室信息管理
- 症状类型管理
- 疾病信息管理
- 就诊信息管理
- 预约挂号管理
- 诊断病历管理
- 评价信息管理
- 预约取消管理
- 在线咨询管理
- 充值信息管理
- 系统管理
- 我的信息

### 医生

- 就诊信息管理
- 预约挂号管理
- 诊断病历管理
- 评价信息管理
- 预约取消管理

- 我的信息
